# Trader-Behavior-Analysis
Analyzed trader performance based on Bitcoin market sentiment (Fear vs Greed). Found that traders perform better during Greed conditions, while Fear leads to lower win rates. Used Python (Pandas, Matplotlib) for data analysis and visualization.
# Trader Behavior Analysis using Market Sentiment

##  Objective
Analyze how trader performance varies based on Bitcoin market sentiment (Fear vs Greed).

##  Datasets Used
- Historical Trader Data (Hyperliquid)
- Fear & Greed Index

##  Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

##  Key Analysis
- Profit comparison between Fear & Greed
- Win rate analysis
- Trading behavior (Buy/Sell)

##  Key Insights
- Traders perform better during Greed markets
- Fear markets show lower win rates
- Market sentiment impacts trading outcomes significantly

##  Conclusion
Understanding sentiment helps improve trading strategies and risk management.
